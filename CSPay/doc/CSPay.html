Running the getData() function without any parameters will scrape all of the data available from payscale for average pay for computer science graduates.

Data for different majors can be scraped my modifying the url parameter to the appropriate url from payscale.com for the specific url you are interested in. You can also modify the number of pages scraped by modifying the pages variable
